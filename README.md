# Samudra Tapu Glacier Mapping

This repository documents my journey of building a reproducible workflow for mapping the Samudra Tapu Glacier in Himachal Pradesh, India using remote sensing, GIS, and machine learning.

The long-term aim is to compare traditional manual glacier mapping methods with modern AI-assisted approaches while developing an open and transparent workflow.

---

## Objectives

- Learn glacier mapping using QGIS
- Study remote sensing fundamentals
- Build a reproducible GIS workflow
- Explore AI-assisted glacier segmentation
- Compare manual and automated glacier outlines
- Document every step of the project

---

## Study Area

**Samudra Tapu Glacier**
- Location: Chandra Basin, Himachal Pradesh, India
- Glacier Type: Debris-covered Himalayan valley glacier
